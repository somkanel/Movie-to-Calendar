# Movie-to-Calendar
使用 Shortcuts/Worflow 将电影票信息导入日历
> 版本：v1.0 [GET](https://www.icloud.com/shortcuts/b1971641b8b141d68757acad21799913)
## 使用方法
1. 复制淘票票购票短信；
2. 在通知中心或捷径应用内运行；
3. 观影信息将自动添加到系统日历。
## 注意事项
- 使用本捷径无需使用网络；
- 本捷径暂时仅支持【淘票票】开头的购票确认短信。
